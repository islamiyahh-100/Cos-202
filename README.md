## 1. Mathematical Calculator (MC)
File: `mathematical_calculator.py`

An interactive calculator supporting:
- Addition (+), Subtraction (-), Multiplication (*), Division (/)
- Power (^), Modulus (%)
- Clear (C) and OFF (exit) commands

The program uses selection control statements (if/elif/else) to determine which operation to perform, and includes error handling for invalid input and division/modulus by zero.

**Sample outputs:** see screenshots in this repository.

## 2. Personal Pocket CGPA Calculator (PPC)
File: `cgpa_calculator.py`

Calculates a student's CGPA on a 5.0 grading scale (A=5, B=4, C=3, D=2, E=1, F=0). For each course entered, the user provides the course code, credit units, and letter grade. The program computes total credit points, total units, final CGPA, and classifies the result (First Class, Second Class Upper, Second Class Lower, Third Class, Pass, or Fail) using selection control statements.

**Sample outputs:** see screenshots in this repository.
